How to set up a Minecraft Server

Getting Started

The Minecraft Server executable requires the Java Runtime Environment to be installed on your computer. If you do not have Java installed, or think you may have an outdated version, you can download the latest version here.

Setting up the server files

Firstly, you will need the server files.

  1. Go to the Minecraft Server Downloads page and download the minecraft_server.<version>.jar file

  2. Create a new folder with a memorable name (e.g. Minecraft Server 1.17) and move the minecraft_server.<version>.jar file into your new folder.

  3. Open the minecraft_server.<version>.jar file with Java by double clicking on it.

You should now see some new files appear in your server directory.

Server Configuration

To host a Minecraft Server, you must agree to the End User License Agreement (EULA)

  1. Inside of your server directory, you should be able to see a eula.txt file. Double click on this file to open it.

  2. Read the contents of the file. Most Importantly, read the Minecraft EULA.

  3. Only if you agree to the EULA, change eula=false to eula=true.

    (If you don't agree to the EULA, stop here and check out Minetest instead!)

  4. Save and close the eula.txt file.

Port Forwarding

In order to allow people from outside your local network to connect to your Minecraft Server, you will need to configure a port forwarding rule in your router's administration dashboard. This step can vary a lot depending on which router you are using.

A Port Forwarding rule will allow people from outside your private local-area network to connect to a specific service on your computer (in this case, a Minecraft server) over the internet, without exposing the rest of your private network to the outside world.

Most household routers will have a web interface that you can connect to via the router's local IP address, which is the default gateway address. To configure a port forwarding rule, first we need to log in to the router web interface.

Finding the Default Gateway address

Windows Instructions

  1. Open the command prompt by holding down the Windows key and press R to bring up the Run dialogue
  2. Type cmd into the Run dialogue and hit Enter
  3. In the command prompt, type in ipconfig and hit Enter
  4. In the console you should be able to see an overview of the Windows IP Configuration. Look for the row with the Default Gateway entry to find your Default Gateway address.

MacOS Instructions

  1. Open up the Terminal application from the Launchpad on MacOS, or a terminal emulator which you prefer.
  2. In your terminal, type in netstat -nr | grep default and hit Enter
  3. You should be able to see your default gateway address at the top of the output.

- If you are on Windows, go onto cmd and type in ‘ipconfig’, look for your IPv4 address. (It should look something like 192.168.0.16) and find your default gateway address (should look something like 192.168.0.1)

- If you are on Mac, go to terminal and type in “ifconfig getifaddr en0” to find your ipv4 address, and then type in “netstat -nr | grep default” to find your default gateway.

- Paste the default gateway address into your web browser url bar, and go.

- If you need to login, the username is almost always admin or administrator, and the password is almost always just ‘password’, or nothing at all.

- Once you are in, look for port forwarding options. This normally comes in under games and services, or firewall settings, if you can’t find it, look for advanced options, and then keep looking.

- Once you have found it, create a new port forwarding rule and call it Minecraft. Change the protocol to both UDP and TCP, and set the WAN and LAN port to 25565. Set the destination IP to your IPv4 address which you found earlier in cmd/terminal

- Save the port forwarding configuration and then exit the web browser.

- Go into the server files and open the server.properties file in a text editor

- Edit the line “server-ip” and make sure it equals your IPv4 address.

- Launch the server

- You can connect to it from the same household via the IPv4 address that you acquired above, but to allow other people to connect to it externally, follow the following steps:

- Go to www.google.com and search for “IP”

- You should be able to find your public IP address

- Send that address to whoever you want to connect to your server, with :25565 at the end of it (It should look something like 60.200.6.202:25565)
